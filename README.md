A beautiful, interactive personal portfolio built with Streamlit showcasing Marje's BSCS journey, hobbies, and contact info! 🍫✨

🌟 Features
🎨 Stunning gradient design with custom CSS & Google Fonts
📱 Fully responsive multi-page layout
🏠 Home page with stats, hobbies, and interactive poll
👤 About page with education background & BSCS subjects
💌 Contact form with email integration & success animations
🎯 Interactive elements - polls, buttons, balloons! 🎈
📊 Quick stats cards for age, year, zodiac
🍫 Personal touches - chocolate & adobo love! 😋
🛠️ Tech Stack
python


streamlit          # Main framework
Custom CSS         # Beautiful styling
Google Fonts       # Dancing Script + Poppins
Multi-page app     # st.switch_page()
Interactive forms  # st.form()
🚀 Quick Start
Prerequisites
Python 3.8+
Streamlit (pip install streamlit)
Installation
bash

# Clone/Download this repo
git clone <your-repo-url>
cd marje-bscs-portfolio

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
